# Oracle Character Platform

## Project Introduction
A web platform for oracle bone inscription image-text retrieval based on Spring Boot and Vue3.

## Tech Stack
- Backend: Spring Boot 3.2 + MyBatis-Plus + MySQL
- Frontend: Vue3 + Vite + Element Plus + Axios + ECharts

## Core Functions
- Character-to-image search
- Image-to-character recognition
- Character detail visualization
